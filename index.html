Chunking

Definition:
Chunking is the process of grouping information into smaller, meaningful units to improve memory and understanding.

Role in HCI:
Chunking helps users:

  • Remember information easily
  • Process complex data efficiently

Examples:

  • Phone number: 0301-123-4567 instead of 03011234567
  • Grouped menu categories instead of one long menu

Benefit:
Chunking reduces cognitive load on users.

Short-Term Memory (STM) in HCI

Definition:
Short-term memory temporarily stores information that is currently being used.

Characteristics:

  • Limited capacity (5‚Äì9 items)
  • Short duration (15‚Äì30 seconds)
  • Easily overloaded

Importance in HCI Design:

  • Interfaces should not rely heavily on user memory
  • Important information should be visible on the screen
  • Systems should provide feedback and reminders

Example:
Auto-filled forms reduce the need for users to remember previous information.
